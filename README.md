# Sports Facility Court Booking Platform

Minimal full-stack implementation (Node/Express + MongoDB + React) with:
- Multi-resource scheduling (court + optional coach)
- Dynamic pricing engine (peak hours, weekend, equipment fee)
- Single localhost in production: Express serves the React build from the same port.



1. Backend dependencies:
   cd backend
   npm install

2. Frontend dependencies:
   cd ../frontend
   npm install

3. Build frontend:
   npm run build

4. Start server:
   npm start

Open http://localhost:5000
